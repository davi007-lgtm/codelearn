HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Define la estructura y el contenido semántico de una página web.

  • Lenguaje de marcado estándar
  • Base de toda página web
  • Estructura semántica
  • Compatible con todos los navegadores
