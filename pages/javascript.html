JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, orientado a objetos, basado en prototipos y multiparadigma. Es uno de los tres pilares de la web moderna junto con HTML y CSS.

  • Dinámico y flexible
  • Ejecución en el navegador
  • Asíncrono y event-driven
  • Gran ecosistema (npm)

Variables y Tipos

Declaración de Variables

En JavaScript moderno, usamos let y const para declarar variables con scope de bloque.

// Variables modernas let nombre = "JavaScript"; const VERSION = "ES2022"; // Scope de bloque { let x = 10; const y = 20; } // Tipos de datos let string = "texto"; let number = 42; let boolean = true; let array = [1, 2, 3]; let object = { key: "value" }; let nulo = null; let indefinido = undefined; // Symbol y BigInt let symbol = Symbol("descripción"); let bigInt = 9007199254740991n;

💡 Usa const por defecto y let solo cuando necesites reasignar valores. Evita usar var.

Tipos y Conversiones

// Verificación de tipos typeof "texto"; // "string" typeof 42; // "number" typeof true; // "boolean" typeof undefined; // "undefined" typeof null; // "object" typeof {}; // "object" typeof []; // "object" // Conversiones String(42); // "42" Number("42"); // 42 Boolean(1); // true Array.from("123"); // ["1", "2", "3"] Object.entries({x: 1}); // [["x", 1]]

⚠️ Cuidado con las conversiones implícitas en comparaciones. Usa siempre === para comparaciones estrictas.

Operadores

Operadores Básicos

// Aritméticos let suma = 5 + 3; // 8 let resta = 10 - 4; // 6 let multiplicacion = 3 * 4; // 12 let division = 15 / 3; // 5 let modulo = 17 % 5; // 2 // Comparación let igual = 5 === "5"; // false (estricto) let igualSuelto = 5 == "5"; // true (no estricto) let mayor = 10 > 5; // true let menor = 3 < 7; // true

Operadores Lógicos

// AND, OR, NOT let and = true && false; // false let or = true || false; // true let not = !true; // false // Operador ternario let edad = 20; let mensaje = edad >= 18 ? "Adulto" : "Menor"; // Operadores de asignación let x = 5; x += 3; // x = x + 3 x *= 2; // x = x * 2
