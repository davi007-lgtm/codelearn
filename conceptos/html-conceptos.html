Conceptos Clave de HTML

Estructura Básica

La estructura básica de un documento HTML5 consiste en:

  • DOCTYPE: Declara el tipo de documento HTML
  • html: Elemento raíz del documento
  • head: Contiene metadatos y enlaces a recursos
  • body: Contiene el contenido visible
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
</head>
<body>
    <!-- Contenido aquí -->
</body>
</html>

Elementos HTML

Los elementos HTML son los bloques de construcción básicos:

  • Elementos de bloque (div, p, h1-h6)
  • Elementos en línea (span, a, strong)
  • Elementos vacíos (img, br, hr)
  • Elementos semánticos (article, section, nav)

HTML Semántico

El HTML semántico da significado estructural al contenido:

<header>
    <nav>
        <!-- Navegación principal -->
    </nav>
</header>

<main>
    <article>
        <section>
            <h1>Título del artículo</h1>
            <p>Contenido...</p>
        </section>
    </article>
    
    <aside>
        <!-- Contenido relacionado -->
    </aside>
</main>

<footer>
    <!-- Pie de página -->
</footer>

Accesibilidad

Prácticas clave para hacer el contenido accesible:

  • Uso de atributos ARIA
  • Textos alternativos para imágenes
  • Estructura jerárquica de encabezados
  • Contraste de colores adecuado
<img src="imagen.jpg" 
     alt="Descripción de la imagen" 
     aria-label="Descripción detallada">

<button aria-label="Cerrar" 
        role="button" 
        aria-pressed="false">
    ×
</button>
